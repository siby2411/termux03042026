Étape 1 : Conception de la Base de Données (Schéma SQL)
Nous devons créer des tables relationnelles pour gérer les utilisateurs, les documents, la messagerie et la facturation/stock.

Copiez et collez ce script SQL dans votre terminal MariaDB (mariadb -u root -p grh) ou via un client comme phpMyAdmin.

SQL
-- 1. Table des Services (Départements)
CREATE TABLE services (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nom_service VARCHAR(50) NOT NULL UNIQUE, -- Comptabilité, Marketing, etc.
    description TEXT
);

-- Insertion des services par défaut
INSERT INTO services (nom_service) VALUES 
('Direction'), ('Comptabilité'), ('Marketing'), ('Secrétariat'), ('Logistique'), ('Informatique');

-- 2. Table des Utilisateurs (Collaborateurs)
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    service_id INT,
    nom VARCHAR(100) NOT NULL,
    prenom VARCHAR(100) NOT NULL,
    email VARCHAR(150) NOT NULL UNIQUE,
    password VARCHAR(255) NOT NULL, -- Hashé (password_hash en PHP)
    role ENUM('admin', 'user', 'manager') DEFAULT 'user',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (service_id) REFERENCES services(id) ON DELETE SET NULL
);

-- 3. Table de Gestion Documentaire (Uploads)
CREATE TABLE documents (
    id INT AUTO_INCREMENT PRIMARY KEY,
    user_id INT, -- Qui a uploadé
    service_cible_id INT, -- Pour quel service (partage)
    nom_fichier VARCHAR(255) NOT NULL,
    chemin_fichier VARCHAR(255) NOT NULL,
    type_mime VARCHAR(50),
    date_upload DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id),
    FOREIGN KEY (service_cible_id) REFERENCES services(id)
);

-- 4. Table de Messagerie Interne
CREATE TABLE messages (
    id INT AUTO_INCREMENT PRIMARY KEY,
    sender_id INT,
    receiver_id INT,
    sujet VARCHAR(200),
    corps TEXT,
    lu BOOLEAN DEFAULT FALSE,
    date_envoi DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (sender_id) REFERENCES users(id),
    FOREIGN KEY (receiver_id) REFERENCES users(id)
);

-- 5. Gestion de Stock (Produits)
CREATE TABLE produits (
    id INT AUTO_INCREMENT PRIMARY KEY,
    designation VARCHAR(200) NOT NULL,
    ref_interne VARCHAR(50) UNIQUE,
    stock_actuel INT DEFAULT 0,
    prix_unitaire DECIMAL(10, 2),
    seuil_alerte INT DEFAULT 10
);

-- 6. Facturation / Commandes
CREATE TABLE commandes (
    id INT AUTO_INCREMENT PRIMARY KEY,
    client_nom VARCHAR(150),
    cree_par_id INT,
    date_commande DATETIME DEFAULT CURRENT_TIMESTAMP,
    etat ENUM('devis', 'validee', 'livree', 'facturee', 'annulee') DEFAULT 'devis',
    total_ht DECIMAL(12, 2),
    FOREIGN KEY (cree_par_id) REFERENCES users(id)
);

-- Optimisation : Index pour accélérer les recherches fréquentes
CREATE INDEX idx_user_email ON users(email);
CREATE INDEX idx_doc_service ON documents(service_cible_id);
CREATE INDEX idx_commande_etat ON commandes(etat);

A. Fichier de connexion (includes/db.php)
PHP
<?php
$host = 'localhost';
$db   = 'grh';
$user = 'root';
$pass = ''; // Votre mot de passe root, vide par défaut en local parfois, sinon mettez le vôtre
$charset = 'utf8mb4';

$dsn = "mysql:host=$host;dbname=$db;charset=$charset";
$options = [
    PDO::ATTR_ERRMODE            => PDO::ERRMODE_EXCEPTION,
    PDO::ATTR_DEFAULT_FETCH_MODE => PDO::FETCH_ASSOC,
    PDO::ATTR_EMULATE_PREPARES   => false,
];

try {
    $pdo = new PDO($dsn, $user, $pass, $options);
} catch (\PDOException $e) {
    throw new \PDOException($e->getMessage(), (int)$e->getCode());
}
?>

Conseils pour l'optimisation (Votre mission d'ingénieur)
Sécurité des Uploads : Dans votre module PHP d'upload, vérifiez toujours l'extension du fichier (autoriser PDF, DOCX, JPG, mais interdire .PHP ou .EXE) pour éviter qu'un utilisateur malveillant n'exécute du code sur le serveur.

Requêtes Préparées : Utilisez toujours PDO::prepare() (comme configuré dans db.php) pour éviter les injections SQL.

Gestion des Sessions : Créez un fichier auth_check.php à inclure en haut de chaque page pour vérifier if (!isset($_SESSION['user_id'])) { header('Location: login.php'); }.
